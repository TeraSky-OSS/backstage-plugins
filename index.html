    Welcome to TeraSky's Backstage Plugin Collection! üöÄ#

    Welcome to our comprehensive collection of Backstage plugins! We've created these plugins to enhance your Backstage experience and make your development workflow more efficient and enjoyable. All our plugins are thoroughly tested and compatible with Backstage version 1.48.0.

    üîß Our Plugin Categories#

    Kubernetes#

    1. Kubernetes Ingestor - Automatically create catalog entities from Kubernetes resources, with support for custom GVKs, Crossplane claims, and KRO resources. It also create GitOps friendly Software Templates for Crossplane Claims/Composites, CRDs, and KRO Instances.
    2. API Docs Module - CRD - Interactive CRD documentation with doc.crds.dev-style visualization, multi-version support, and example YAML generation.
    3. Crossplane Suite - Complete Crossplane integration for cloud resource management.
    4. KRO Suite - Complete KRO integration for Kubernetes resource orchestration.
    5. Kyverno Policy Reports - Policy compliance monitoring for Kubernetes.
    6. Kubernetes Resources - Visual graph representation of Kubernetes resources and their dependencies with simple Backstage permission and RBAC management.
    7. SpectroCloud Suite - Complete SpectroCloud Palette integration with catalog ingestion, frontend visualization, Kubernetes cluster discovery, OIDC authentication for single sign-on and user-level cluster access, and MCP actions for AI agents.

    Infrastructure As Code#

    1. GitOps Manifest Updater - Update GitOps manifests and create PRs with ease.
    2. Terraform Scaffolder - Generate Input forms in software templates for terraform module variables.

    Training & Education#

    1. Educates Platform - Complete training platform with workshop management.

    Development Tools#

    1. DevPod Plugin - Quick development environment setup with IDE integration.
    2. Spring Initializer - Create Spring Boot applications with smart dependency filtering, version compatibility checking, and admin-configurable defaults.

    Resource And Cost Optimization#

    1. ScaleOps Suite - Complete ScaleOps integration for cost optimization with AI-powered automation through MCP actions.

    Cloud Infrastructure#

    1. VCF Automation Suite - Comprehensive VMware Cloud Foundation Automation integration.
    2. VCF Operations Suite - Comprehensive VMware Cloud Foundation Operations observability integration.

    AI#

    1. AI Coding Rules - Comprehensive visualization and management of AI coding rules from Cursor, GitHub Copilot, and Cline.
    2. Scaffolder MCP Backend - MCP actions for AI agents to discover, inspect, and execute software templates programmatically.
    3. RBAC MCP Backend - MCP actions for AI agents to manage roles, permissions, and access policies programmatically.
    4. Catalog MCP Backend - MCP actions for AI agents to query and discover catalog entities programmatically.

    General Utilities#

    1. Scaffolder Actions - Powerful actions for automation:
      • Crossplane claim template creation
      • CRD template creation
      • Catalog info cleaning
    2. Entity Scaffolder Plugin - Embbed scaffolder templates in component pages.
    3. Template Builder - Visual WYSIWYG editor for creating Backstage Software Templates with drag-and-drop workflow canvas, intelligent YAML autocomplete, and automatic field extension discovery.
    4. Global Sign-In Page - Configuration-driven sign-in page supporting all core authentication providers through YAML config.
