# template-builder

Welcome to the Template Builder plugin!

[![npm latest version](https://img.shields.io/npm/v/@terasky/backstage-plugin-template-builder/latest.svg)](https://www.npmjs.com/package/@terasky/backstage-plugin-template-builder)

The `template-builder` plugin for Backstage provides a visual, drag-and-drop WYSIWYG editor for creating and editing Backstage Software Templates. It features an interactive workflow canvas with React Flow, real-time YAML synchronization with Monaco editor, intelligent autocomplete for actions and parameters, and a comprehensive input designer for all field types including custom field extensions. The editor supports template validation, automatic graph layout (horizontal/vertical), and allows downloading templates as YAML files.

For detailed docs go to https://terasky-oss.github.io/backstage-plugins/plugins/template-builder/overview
