# Scaffolder MCP Backend

Welcome to the scaffolder-mcp-backend plugin!

[![npm latest version](https://img.shields.io/npm/v/@terasky/plugin-scaffolder-mcp-backend/latest.svg)](https://www.npmjs.com/package/@terasky/plugin-scaffolder-mcp-backend)

## Description

The `scaffolder-mcp-backend` plugin for Backstage provides Model Context Protocol (MCP) actions for interacting with Backstage software templates. This plugin enables AI agents and automation tools to discover, inspect, and execute software templates programmatically.

For detailed docs go to https://terasky-oss.github.io/backstage-plugins/plugins/scaffolder-mcp/overview
