# spectrocloud-auth-backend

Welcome to the spectrocloud-auth-backend plugin!

[![npm latest version](https://img.shields.io/npm/v/@terasky/backstage-plugin-spectrocloud-auth-backend/latest.svg)](https://www.npmjs.com/package/@terasky/backstage-plugin-spectrocloud-auth-backend)

The `spectrocloud-auth-backend` module for Backstage provides SpectroCloud OIDC authentication integration with proper token exchange, enabling secure single sign-on and user-level Kubernetes cluster access. This module implements the OAuth2 authorization code flow with client secret authentication to obtain RS256-signed ID tokens from SpectroCloud.

For detailed docs go to https://terasky-oss.github.io/backstage-plugins/plugins/spectrocloud/auth/overview
