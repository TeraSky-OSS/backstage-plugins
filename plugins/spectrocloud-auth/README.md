# spectrocloud-auth

Welcome to the spectrocloud-auth plugin!

[![npm latest version](https://img.shields.io/npm/v/@terasky/backstage-plugin-spectrocloud-auth/latest.svg)](https://www.npmjs.com/package/@terasky/backstage-plugin-spectrocloud-auth)

The `spectrocloud-auth` plugin for Backstage provides the frontend OAuth2 client implementation for SpectroCloud OIDC authentication. It creates the necessary API references and OAuth2 client that enable sign-in functionality, session management, and integration with the sign-in page.

For detailed docs go to https://terasky-oss.github.io/backstage-plugins/plugins/spectrocloud/auth/overview
