# vcfsso-auth

Welcome to the VCF SSO authentication frontend plugin!

[![npm latest version](https://img.shields.io/npm/v/@terasky/backstage-plugin-vcfsso-auth/latest.svg)](https://www.npmjs.com/package/@terasky/backstage-plugin-vcfsso-auth)

The `vcfsso-auth` plugin for Backstage provides the OAuth2 client implementation for VCF SSO OIDC authentication. It creates the necessary API reference and OAuth2 client that enable sign-in functionality and session management with VCF SSO as the identity provider.

For detailed docs go to https://terasky-oss.github.io/backstage-plugins/plugins/vcf-automation/vcfsso/frontend/about
