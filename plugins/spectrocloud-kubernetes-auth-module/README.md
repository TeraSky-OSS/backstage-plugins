# spectrocloud-kubernetes-auth-module

Welcome to the spectrocloud-kubernetes-auth-module plugin!

[![npm latest version](https://img.shields.io/npm/v/@terasky/backstage-plugin-spectrocloud-kubernetes-auth-module/latest.svg)](https://www.npmjs.com/package/@terasky/backstage-plugin-spectrocloud-kubernetes-auth-module)

The `spectrocloud-kubernetes-auth-module` for Backstage is a frontend module that extends the Kubernetes plugin to add SpectroCloud OIDC authentication support. It enables user-level authentication to Kubernetes clusters using SpectroCloud identity tokens, providing audit trails and eliminating the need for service account management.

For detailed docs go to https://terasky-oss.github.io/backstage-plugins/plugins/spectrocloud/auth/overview
